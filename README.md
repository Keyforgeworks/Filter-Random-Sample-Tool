# Filter and Random Sample Tool

## Description

PowerShell tool that provides filtering and random sampling data. Efficiently filter datasets based on specified criteria and then extract a random subset of the filtered results. This tool is designed to streamline data analysis tasks by providing a convenient way to isolate specific data and create representative samples for further investigation. The refactored "open file" button functionality enhances the user experience.

## Features

*   Filtering of data based on user-defined criteria.
*   Random sampling of filtered data.

## Installation

1.  Ensure you have PowerShell installed on your system.
2.  Download the executable file (.exe) from the [Releases](https://github.com/keyforgeworks/Filter-Random-Sample-Tool/releases) section of this repository.
3.  Place the executable file in a directory of your choice.

## Usage

1.  Run with PowerShell or run the executable directly from compiled releases.
2.  Follow the on-screen prompts to filter and sample your data.
